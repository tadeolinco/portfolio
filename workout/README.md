# Workout

Just a workout tracker with in-built timer


### Motivations

1. I just wanted a timer inside the usual workout apps

### Tech stack
1. NextJS - FE and BE APIs - all-in-one framework, easy integration with vercel
2. Prisma - ORM, generates types
3. Supabase - DB and Auth, Firebase alternative for postgres, generous free-tier with dashboard


### How to run dev

1. `yarn install`
2. `cp .env.sample .env`
3. Fill in `.env`
4. `yarn dev`
