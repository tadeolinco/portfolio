# Gastos

Just a day to day financial transctions tracker

### Motivations

1. I didn't need a budget app, as the main purpose was not to budget but simply to track
2. Be able to put currency of transaction easily
3. Be able to see converions of transaction easily (hard-coded tho)

### Tech stack
1. NextJS - FE and BE APIs - all-in-one framework, easy integration with vercel
2. Prisma - ORM, generates types
3. Supabase - DB and Auth, Firebase alternative for postgres, generous free-tier with dashboard


### How to run dev

1. `yarn install`
2. `cp .env.sample .env`
3. Fill in `.env`
4. `yarn dev`
